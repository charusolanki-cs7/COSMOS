# Cosmos AI Assistant

Cosmos is a desktop AI assistant built using Python that can perform task such as opening desktop applications. This project is designed to be versatile and extensible, with the ability to add more functionalities easily.

## Features

**Voice Activation:** Activate Cosmos by saying "Cosmos."
**Text Input:** Type your queries and press enter to receive a response.
**App Control:** Open applications like Notepad and OneNote.
**Website Navigation:** Open websites like YouTube and Canva.
**Multimedia Control:** Search and play specific videos on YouTube.
**Phone and WhatsApp Communication:** Make calls or send messages.

## Usage

### Activating the Assistant

There are several ways to activate Cosmos:

* Voice Activation: Say "Cosmos."
* Text Input: Type your query (e.g., "How are you?")
* Keyboard Shortcut: Press Window + J
